# [Monado Sword](https://xenoblade.fandom.com/wiki/Monado)

> giao
 
MONADO 名字来源于游戏《异度神剑》是其中由主角使用的武器，神剑 MONADO 传说由神赐予**ホムス族**的圣剑。这实际上并不是一把剑，
而是一个藏着某种力量的剑型“装置”，能于战斗时变为光之刃的形态。


## usage

```shell
# 初始化项目
npx @monado/cli create <projectName>

# 推荐使用yarn安装依赖
npm install -g yarn

# 安装依赖
yarn

# 启动开发服务
yarn start

# 打包
yarn build
```
